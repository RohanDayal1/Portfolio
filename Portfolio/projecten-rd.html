Project 1

Ouderavond Website

Voor de herfstvakantie ben ik met mijn klasgenoten naar school gegaan om aan onze ouders/verzorgers te vertellen over onze nieuwe opleiding tijdens de ouder-informatieavond. De ouder-informatieavond bestond uit twee delen: In het eerste deel hebben we met een klasgenoot onze wederzijdse ouders verteld over hoe de organisatie van onze opleiding in elkaar zit. We hebben dit gedaan aan de hand van een door onszelf gebouwde website met HTML en CSS.
In het tweede deel van de avond hebben ik met mijn project partner de ouders verteld over het informatie systeem van onze school genaamd EduArte aan de hand van een powerpoint presentatie.

Project 2

Game Website

Ik heb in periode 2 gewerkt aan een project dat deels bestond uit herhaling en verdieping van HTML-CSS en deels uit het toepassen van de kennis van Javascript uit periode 1 en 2. Samen met twee klasgenoten heb ik een website gebouwd waarin we onze zelfgebouwde Javascript games hebben opgenomen.
Het resultaat was een website met een homepage en per kandidaat 4 pagina's, namelijk een portfolio, beschrijving van onze favoriete games, uitleg van de zelfgebouwde game en een pagina om de zelfgebouwde game te spelen. De javascript game staat ook op deze pagina om te bekijken

Project 3

Webshop

Ik ben bezig met het bouwen van een dynamische website voor het project van deze periode met behulp van PHP en MYSQL. Het dynamische deel van de website zal gegevens tonen uit een database, waarbij de informatie op de website verandert wanneer de inhoud van de database wordt gewijzigd. Samen met mijn projectteam heb ik gekozen voor een bedrijf dat producten of diensten verkoopt.
Op dit moment zijn we bezig met het uitbreiden van de website door formulieren toe te voegen voor het indienen van klachten of suggesties over de website of producten, inclusief een response-pagina. Ook zullen we een serie pagina's waarop gegevens aan de database worden toegevoegd. Naast dit werk ben zijn we ook bezig met het ontwerpen, bouwen en vullen van de achterliggende database.
